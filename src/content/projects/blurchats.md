Built a Chrome browser extension from scratch that programmatically blurs Gemini chat sidebars, giving users a privacy overlay to prevent shoulder-surfing without closing conversations.
Implemented a content script using the Chrome Extensions Manifest V3 API to inject CSS blur filters into live DOM elements, achieving seamless privacy toggling with zero page refresh required
